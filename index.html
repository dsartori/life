John Conway's Game Of Life

Overview

This is an implementation of John Conway’s Game of Life. In Life, “cells” live on a two-dimensional grid and survive, reproduce, and die based on four rules:

  • Any live cell with fewer than two live neighbours dies.
  • Any live cell with more than three live neighbours dies.
  • Any live cell with two or three live neighbours lives on.
  • Any dead cell with exactly three live neighbours becomes a live cell.

Controls

Click or tap a cell on the board to toggle it's state. The board is larger than the view window. Scroll using gestures or the scroll bars. Select a starting pattern to use from the dropdown box.

  • Advance the simulation one step.
  • Play the simulation forward at 1x speed
  • Play the simulation forward at 4x speed
  • Pause the simulation
  • Load the selected board setup
  • Toggle this help screen
